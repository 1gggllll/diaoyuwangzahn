## 一、课题要交什么（大纲最低要求）

| 序号 | 要求 | 对应文件 |
|------|------|----------|
| 1 | 游戏吸引页 | `index.php`（英雄联盟限定福利） |
| 2 | 登录 + 跳转 + 数据库 | `login.php` → `submit.php` → `success.php` |
| 3 | URL 欺骗或二维码 | `url_demo.php`、`qrcode.php`（`?from=url` / `?from=qrcode`） |

| 5 | PHP 实现 | 全部 PHP |
| + | 课题名「防护」 | `defense.php` + `login.php?test_block=1` |

**课设不做**：Git、Nginx、CSRF、限流；**密码按明文入库**以演示窃取效果（仅授权环境）。

## 四、环境搭建

1. 安装 XAMPP 或使用 `.tools\php`  
2. 导入 `init.sql`  
3. 配置 `config.local.php`（MySQL 密码、`$demo_base` 可选）  
4. 双击 `start-phishguard.bat`  
5. 访问首页验收

## 五、核心要点

### channel 来源

- `index.php` / `login.php`：`?from=direct|url|qrcode` → `$_SESSION['channel']`  
- `submit.php`：**Session 优先**  
- URL 渠道演示：打开 `index.php?from=url`（非 url_demo 内链）  
- 二维码：`qrcode.php` 使用 `demo_base_url()` 生成可扫地址  

### 拦截演示

`login.php?test_block=1` → `defense.php?blocked=1`

### 后台

- `admin/admin123`  
- 列表明文显示 password；3 秒 meta refresh
